Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM :
  • By ID - document.getElementById()
  • By Class Name - document.getElementsByClassName()
  • By Tag Name - document.getElementsByTagName()
  • By Name Attribute - document.getElementsByName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The key differences between innerHTML, innerText, and textContent is :
  • innerHTML - Retrieves or sets HTML content, including tags and nested elements.
  • innerText - Retrieves or sets the visible text of an element.
  • textContent - Retrieves or sets the entire text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element to manage events on multiple child elements, even those added dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event triggered on a child element propagates (or "bubbles up") to its parent and ancestors, starting from the target element up to the document.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Creating Elements - Use document.createElement(tagName) to create an element.
  • Adding Elements to the DOM -
  • Append as Last Child (appendChild).
  • Insert Before Another Element (insertBefore).
  • Insert at a Specific Position (insertAdjacentHTML).
  • Removing Elements - Using removeChild().
